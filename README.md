# 🎯 Java Playground

Java 언어의 기초부터 고급 기능까지 학습하고 실습하는 개인 학습 저장소입니다.

## 📚 학습 목표

- Java 기본 문법 및 객체지향 프로그래밍 완벽 이해
- JVM 동작 원리와 메모리 관리 학습
- 실무에서 사용되는 디자인 패턴 구현
- 모던 Java (8~21) 기능 활용
